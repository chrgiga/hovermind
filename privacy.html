HoverMind Privacy Policy

Last updated: March 2026

1. Data Collection

HoverMind ("the Extension") does not collect, transmit, distribute, or sell any personal data of the user to developers or third parties.

2. Local Storage

All information necessary for the operation of the Extension, including API keys, user preferences, model configuration, and history of cached texts, is stored locally and encrypted on the user's device using native browser tools (Chrome Storage).

3. Interaction with Third-Party Services

When the user selects text and requests a definition or translation, the Extension sends that text directly to the APIs of the providers configured by the user (such as OpenAI, Google Gemini, Anthropic, DeepSeek, or MyMemory). The treatment of that information is subject to the individual privacy policies of those providers. The Extension does not act as an intermediary or store these communications outside of the local device.

4. Changes to the Privacy Policy

Any future changes to this policy will be reflected on this same page and notified in the Extension release notes.
